# 命理师校准案例库

收集自段建业、徐伟刚、《子平真诠》、《滴天髓》、黄勇智等实战命理师的真实案例分析。用于 AI 报告校准。

## 案例 1：丙火日主 · 身强 · 火炎土燥
**来源**：徐伟刚《八字实例详解》
**人物**：当代易学大家（隐名）
**八字**：癸巳 乙丑 丙午 甲午
**日主**：丙火
**大运**：庚寅 辛卯 壬辰 癸巳 **甲午** 乙未 丙申
**旺衰判断**：腊月丙火身已不算弱，"气进两阳"，不以旺衰为急切问题
**病处**：年月三重土（己丑月 + 辰年）晦光 + 寒湿阴土
**用神**：甲木、乙木、寅木、卯木（制己丑湿土）
**忌神**：己土、丑土
**分析要点**：
- 丙火不惧壬癸水官杀，最忌戊己土晦火
- 乙卯字伏制己丑土最有力；甲寅克合己丑过于"有情"略乏力
- 甲午运（42岁后）为人生转折，调入出版社
- 乙未运（52-62岁）成名：1994甲戌成立研究会，1999己卯出版《神奇之门》
- 命中四凶神皆有，一生病灾

## 案例 7：辛金日主 · 金水伤官
**来源**：段建业《盲派速成教程》
**八字**：丙戌 戊戌 辛巳 癸巳
**日主**：辛金
**格局**：不看旺衰，火与燥土一片
**做功**：时柱癸巳自合，制食神癸水。巳之原身戊透月合癸
**分析**：食神虚透在门户被制，主口才。辛金配癸水，搞文化工作。曾在某大学当教师

## 案例 8：辛金日主 · 丙辛合 · 禄被合坏
**来源**：段建业《盲派速成教程》
**人物**：玛丽莲·梦露
**八字**：丙寅 癸巳 辛酉 癸巳
**日主**：辛金
**做功**：辛坐酉禄连根，支中两巳合坏此根（不吉）
**特点**：金水相生主秀气，癸水与坐下巳自合（官星），招男人喜爱
**大运**：丙寅 乙卯 甲寅 癸丑 壬子 辛亥 庚戌 己酉 戊申 丁未
**应期**：大运走寅，酉之绝地，巳增旺，酉被合绝，死于此运

## 核心校准维度

用以对比 AI 输出的关键检查点：
1. 旺衰判断（得令/得地/得生/得助）
2. 用神取法（扶抑/调候/通关/病药）
3. 格局判别（正格/从格/专旺）
4. 十神解读（最深十神对性格的塑造）
5. 大运流年应期（五行能量转换）
